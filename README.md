## Install
```bash
git clone https://github.com/fetsare/Bernstein-Vazirani-in-Qiskit
cd Bernstein-Vazirani-in-Qiskit
```

```bash
python -m venv venv
```

* On Linux/macOS:
```bash
source venv/bin/activate
```
* On Windows (PowerShell):

```powershell
.\venv\Scripts\Activate.ps1
```

```bash
pip install -r requirements.txt
```
